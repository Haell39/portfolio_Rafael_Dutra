## How to Run

1. **Install Dependencies** (if you haven't already):
   ```bash
   npm install
   ```

2. **Start Development Server**:
   ```bash
   npm run dev
   ```
   Visit `http://localhost:3000` to see it in action.

3. **Build for Production**:
   ```bash
   npm run build
   ```
